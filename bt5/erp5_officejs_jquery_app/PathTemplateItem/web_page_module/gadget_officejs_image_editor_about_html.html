OfficeJS Image Editor

OfficeJS Image Editor app is a progressive web application developed with renderJS and jIO under the premises of privacy and availability, meaning users are free to choose their storage, can work offline and synchronize their work easily. Image editor application is developed, maintained and used by Nexedi, and its code can be found here.

The editor itself is based on miniPaint software. The source code of miniPaint can be found here.
