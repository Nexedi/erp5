OfficeJS Web Table Editor

OfficeJS Web Table Editor app is a progressive web application developed with renderJS and jIO under the premises of privacy and availability, meaning users are free to choose their storage, can work offline and synchronize their work easily. Web Table editor application is developed, maintained and used by Nexedi, and its code can be found here.

The editor itself is based on Jspreadsheet software. The source code of Jspreadsheet can be found here.
