chart_introspection_interface

List of chart introspection interfaces which can let gadget to introspect the inside of a chart.

The purpose of this interface definition is to provide an unique API for various charting libraries to introspect the chart itself. Then other gadget can provide these functions to introspect the chart.

Suppose a gadget want to a value of a specific item in a chart. It can aquire the correspond function by declareAcquiredMethod. In this case, it is getChartItemValue. Chart wrapper gadget need to provide this function by using allowPublicAcquisition.

For now, the chart item was identified by name field. This need to do more investivation, For example, I'm not sure that if all chart libraries support to assign different names for all items in the combined-bar chart.
