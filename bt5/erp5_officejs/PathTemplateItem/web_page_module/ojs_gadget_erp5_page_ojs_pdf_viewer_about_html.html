OfficeJS PDF Viewer

OfficeJS PDF Viewer app is a progressive web application developed with renderJS and jIO under the premises of privacy and availability, meaning users are free to choose their storage, can work offline and synchronize their work easily. PDF Viewer application is developed, maintained and used by Nexedi, and its code can be found here.

The PDF Viewer itself is based on PDFjs software. The source code of PDFjs can be found here.
