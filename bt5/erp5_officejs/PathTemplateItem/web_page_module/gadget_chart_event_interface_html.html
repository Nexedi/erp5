chart_event_interface(Draft)

List of chart events interface which can reponse the correspond events in a chart.

The purpose of this interface definition is to provide an unique API for various charting libraries event processing. Then other gadget can provide these function to reponse chart's events.

Suppose a user use mouse to click one item in a chart. If another gadget want to reponse that event. It need to provide the correspond function by allowPublicAcquisition. In this case, the function is chartItemClicked. Chart gadget will pass parameters to this function. Then the gadget can do some custom processing with the parameters in this function.

chartItemClicked
Cilck an item in a chart.
event_params
Schema available here : gadget_chart_event_interface_html_schema.json
One item in a chart was clicked. Any gadget which defined this function can do some custom processing to response that event. Please see json schema for supported parameters.
