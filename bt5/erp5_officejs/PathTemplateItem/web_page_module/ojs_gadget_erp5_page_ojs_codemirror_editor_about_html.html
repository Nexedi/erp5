OfficeJS JS Scripts Editor

OfficeJS JS Scripts Editor app is a progressive web application developed with renderJS and jIO under the premises of privacy and availability, meaning users are free to choose their storage, can work offline and synchronize their work easily. JS Scripts editor application is developed, maintained and used by Nexedi, and its code can be found here.

The editor itself is based on Codemirror software. The source code of Codemirror can be found here.
