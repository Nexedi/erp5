NMS User Learning Track

This learning track is meant for USERS of a Network Management System (NMS) based on SlapOS. After finishing this learning track, you should have a good enough understanding of how to use the NMS to manage a network.

The track is split into lectures introducing a certain component and showing how they are setup and fit into the overall architecture.

Table of Content

  • Lecture 0: Introduction
  • Lecture 1: How to add LTE Service
  • Lecture 2: How to create a new Software Release
  • Lecture 3: How to update configuration of eNB or EPC
  • Lecture 4: How to add User to SimcardDB

Lecture 0: NMS Introduction

This section is a global introduction to NMS and only contains reading materials. It will introduce the NMS and underlying SlapOS sytems and provide answers to key questions. The following documents should be read:

  • OSTV Design Document - OSVT NMS Introduction (presentation)

Lecture 1: How to add LTE Service

This section will show how to instantiate a new LTE service (eNB or EPC) on a machine. It does NOT cover setting up a machine itself or adding the Amarisoft LTE software release to the software catalog of the SlapOS Master so it can be installed on this machine. These steps are part of the admin learning track and can be found in installing COMP-123 LTE Box (presentation). The following document should be read:

  • OSTV HowTo - Add LTE Service (presentation)

Lecture 2: How to create a new Software Release

This section will show how to create a basic software release and explain how the different parts interact. The result of this howto will be a software profile (software.cfg) file. If put on a repository with accessible url, this profile can be used in a NMS network.

This howto does NOT cover how to add a new software product and this release to the NMS software catalogue or how to install this software on a network node. These steps can be found in Install SlapOS Node (COMP-0) (presentation) and installing a network node (COMP-123) (presentation). The following document should be read for this lecture:

  • OSTV HowTo - Create Software Release (presentation)

Lecture 3: How to update configuration of eNB or EPC

This section will show how to update the configuration of an eNB or EPC instance through the NMS Dashboard. It does NOT cover adding new parameters to eNB or EPC themself, which is shown in the lecture on extending the software release (presentation). The following document should be read for this lecture:

  • OSTV HowTo - Update eNB or EPC (presentation)

Lecture 4: How to add User to SimcardDB

This section will show how to instantiate a new user (SimcardID) on the SimcardDB. The following document should be read:

  • OSTV HowTo - Add User to SimcardDB (presentation)
