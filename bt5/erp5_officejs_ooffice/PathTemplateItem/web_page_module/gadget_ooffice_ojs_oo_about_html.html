OfficeJS Only Office Editor

OfficeJS Only Office apps are progressive web applications developed with renderJS and jIO under the premises of privacy and availability, meaning users are free to choose their storage, can work offline and synchronize their work easily. Only Office applications are developed, maintained and used by Nexedi, and their code can be found here.

The editor itself is based on a modified version of OnlyOffice code with patches to support offline operation. The original source code of onlyoffice can be found here and the patched version can be found here.
