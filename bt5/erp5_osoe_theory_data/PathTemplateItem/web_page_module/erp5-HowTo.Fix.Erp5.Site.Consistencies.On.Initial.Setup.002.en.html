ERP5 Site initial setup

by OSOE Project.

This guide will teach you how to handle promises and check site consistency

Access your ERP5 and log in

Access your ERP5 instance by using the "login" and password provided by SlapOS.

Check Site Consistency

In "My Favourites" menu, select "Check Site Consistency".

Fix inconsistencies

Solve site inconsistencies by selecting all and clicking on "Fix Site Configuration".

Wait for activites runs

It can take few minutes, but some modules should be showed on front page. Otherwise you can repeat the process by going to "Check Site Consistency"

After this initial setup you can now start to develop on your instance and/or use configurator for configuring your ERP5 for the end user.
