Agenda

  • Create Data Product

 

This tutorial teaches how to create Data Product.

Data Product represents the type of data. 

In this tutorial we will create a Data Product for incoming raw data from streaming hardware. 

Home

 

Go to your wendelin instance homepage and click on Modules.

Data Product

 

On the modules page click on Data Product.

 

Add

 

Click on Add button.

 

Create Document

 

Continue with clicking on Create Document.

 

Fill the Form

 

Fill the form and click save.

 

Validate

 

After it is saved, click on Validate on the left side panel to validate the Data Product.

 

Confirm Valdation

 

Click on Proceed to confirm the validation.
